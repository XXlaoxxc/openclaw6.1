# 第7章：文件与目录权限

## 📖 核心知识点

### 7.1 权限说明
```
-rwxr-xr-x. 1 root root 1234 Jan 1 12:00 file
│├─├─├─┤
│ │  │  └─ 其他用户权限 (r-x)
│ │  └──── 所属组权限 (r-x)
│ └─────── 所有者权限 (rwx)
└───────── 文件类型 (-:文件 d:目录 l:链接)
```

### 7.2 权限含义
| 权限 | 文件 | 目录 |
|------|------|------|
| r (读) | 查看内容 | 列出目录内容 |
| w (写) | 修改内容 | 增删文件 |
| x (执行) | 运行文件 | 进入目录 |

### 7.3 权限管理命令
| 命令 | 功能 |
|------|------|
| chmod | 修改权限 |
| chown | 修改所有者 |
| chgrp | 修改所属组 |
| umask | 默认权限掩码 |

### 7.4 chmod 用法
```bash
# 数字方式
chmod 755 file    # rwxr-xr-x
chmod 644 file    # rw-r--r--

# 符号方式
chmod u+x file    # 所有者加执行权限
chmod g-w file    # 组去掉写权限
chmod o=r file    # 其他用户只读
```

## 🔑 关键概念

- **SUID/SGID**：特殊权限，执行时继承属主/属组权限
- **Sticky Bit**：目录中只有文件所有者可删除
- **ACL**：访问控制列表，更细粒度的权限控制
