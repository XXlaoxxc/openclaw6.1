# 第5章：首次登录与在线求助

## 📖 核心知识点

### 5.1 登录流程
```
1. 开机 → BIOS → GRUB2 → 内核 → systemd
2. 显示登录界面 → 输入用户名/密码
3. 登录成功 → 进入 Shell
```

### 5.2 基础命令
| 命令 | 功能 |
|------|------|
| date | 显示日期时间 |
| cal | 显示日历 |
| who | 查看在线用户 |
| hostname | 查看主机名 |
| uname | 查看系统信息 |
| clear | 清屏 |

### 5.3 在线求助
| 方式 | 命令/说明 |
|------|-----------|
| man | `man ls` 查看 ls 手册 |
| info | `info ls` 查看 info 文档 |
| --help | `ls --help` 简要帮助 |
| type | `type ls` 查看命令类型 |

### 5.4 man 手册章节
| 章节 | 内容 |
|------|------|
| 1 | 用户命令 |
| 2 | 系统调用 |
| 3 | 库函数 |
| 4 | 设备文件 |
| 5 | 配置文件格式 |
| 8 | 管理员命令 |

## 🔑 关键概念

- **root 用户**：超级管理员，UID=0
- **普通用户**：权限受限，UID≥1000
- **PATH 变量**：命令搜索路径
