# 第8章：目录结构配置（FHS）

## 📖 核心知识点

### 8.1 FHS 标准
Filesystem Hierarchy Standard，定义 Linux 目录结构

### 8.2 重要目录
| 目录 | 说明 |
|------|------|
| / | 根目录，一切起点 |
| /bin | 基本用户命令 |
| /sbin | 系统管理命令 |
| /etc | 配置文件 |
| /dev | 设备文件 |
| /proc | 进程信息（虚拟文件系统） |
| /sys | 硬件信息（虚拟文件系统） |
| /var | 可变数据（日志、缓存） |
| /tmp | 临时文件 |
| /usr | 用户程序和数据 |
| /opt | 第三方软件 |
| /home | 普通用户家目录 |
| /root | root 用户家目录 |
| /boot | 启动文件（内核、引导） |
| /lib | 库文件 |
| /mnt、/media | 挂载点 |

### 8.3 虚拟文件系统
- **/proc**：进程和内核信息
- **/sys**：设备和驱动信息
```bash
cat /proc/cpuinfo    # CPU 信息
cat /proc/meminfo    # 内存信息
```

## 🔑 关键概念

- **一切皆文件**：Linux 中设备、进程都用文件表示
- **挂载点**：分区关联到目录树的位置
