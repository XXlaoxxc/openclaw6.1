# 第9章：文件与文件系统

## 📖 核心知识点

### 9.1 文件类型
| 类型 | 标识 | 说明 |
|------|------|------|
| 普通文件 | - | 文本、二进制等 |
| 目录 | d | 包含其他文件 |
| 链接 | l | 快捷方式 |
| 块设备 | b | 硬盘等 |
| 字符设备 | c | 终端等 |
| 管道 | p | 进程间通信 |
| 套接字 | s | 网络通信 |

### 9.2 inode 与 block
- **inode**：存储文件元数据（权限、大小、时间、block位置）
- **block**：存储文件实际内容
- 一个文件 = inode + 多个 block

### 9.3 文件系统类型
| 类型 | 特点 |
|------|------|
| ext4 | Linux 经典文件系统 |
| xfs | 高性能，CentOS 7 默认 |
| btrfs | 新一代，支持快照 |
| swap | 交换分区 |

### 9.4 磁盘管理命令
| 命令 | 功能 |
|------|------|
| df | 查看文件系统使用情况 |
| du | 查看目录/文件大小 |
| fdisk | 分区工具 |
| mount/umount | 挂载/卸载 |
| fsck | 文件系统检查 |

### 9.5 硬链接与软链接
```bash
ln file link        # 硬链接（共享 inode）
ln -s file link     # 软链接（类似快捷方式）
```

## 🔑 关键概念

- **inode 号**：文件唯一标识
- **block 大小**：通常 4KB
- **挂载**：将文件系统关联到目录树
