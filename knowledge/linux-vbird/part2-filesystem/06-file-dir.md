# 第6章：文件与目录管理

## 📖 核心知识点

### 6.1 目录结构
- **绝对路径**：从根目录开始 `/home/user`
- **相对路径**：从当前目录开始 `./file`、`../dir`

### 6.2 目录操作命令
| 命令 | 功能 | 常用选项 |
|------|------|----------|
| cd | 切换目录 | `cd ~` 回家，`cd -` 回上次 |
| pwd | 显示当前路径 | |
| mkdir | 创建目录 | `-p` 递归创建 |
| rmdir | 删除空目录 | |
| ls | 列出内容 | `-l` 详细，`-a` 隐藏 |

### 6.3 文件操作命令
| 命令 | 功能 | 常用选项 |
|------|------|----------|
| cp | 复制 | `-r` 递归，`-p` 保留属性 |
| mv | 移动/重命名 | |
| rm | 删除 | `-r` 递归，`-f` 强制 |
| touch | 创建空文件/更新时间戳 | |
| cat | 查看文件内容 | `-n` 显示行号 |
| more/less | 分页查看 | |
| head/tail | 查看头尾 | `-n` 行数 |

### 6.4 文件搜索
| 命令 | 功能 |
|------|------|
| find | 按条件搜索 |
| locate | 按数据库搜索 |
| which | 搜索命令路径 |
| whereis | 搜索相关文件 |

## 🔑 关键命令详解

```bash
# ls 常用组合
ls -la          # 详细列表，含隐藏文件
ls -lh          # 人类可读大小
ls -lS          # 按大小排序

# cp 示例
cp -r dir1 dir2 # 复制目录
cp -p file1 file2  # 保留时间戳和权限

# rm 警告
rm -rf /        # ⚠️ 危险！删除根目录
rm -rf *        # ⚠️ 危险！删除当前所有
```
