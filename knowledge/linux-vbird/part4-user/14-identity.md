# 第14章：用户身份与群组

## 📖 核心知识点

### 14.1 用户身份
- **UID**：用户唯一标识数字
  - root：UID = 0
  - 系统用户：UID 1-999
  - 普通用户：UID ≥ 1000

### 14.2 关键文件
| 文件 | 内容 |
|------|------|
| /etc/passwd | 用户信息（用户名:x:UID:GID:描述:家目录:Shell） |
| /etc/shadow | 用户密码（加密） |
| /etc/group | 群组信息 |

### 14.3 /etc/passwd 格式
```
root:x:0:0:root:/root:/bin/bash
│   │ │ │ │    │     │
│   │ │ │ │    │     └─ 登录 Shell
│   │ │ │ │    └─────── 家目录
│   │ │ │ └──────────── GID（主要群组）
│   │ │ └────────────── UID
│   │ └──────────────── 密码占位（x 表示在 shadow 中）
│   └────────────────── 用户名
└────────────────────── 用户说明
```

### 14.4 有效群组与初始群组
- **初始群组**：用户登录时的群组（/etc/passwd 中的 GID）
- **有效群组**：当前使用的群组（`newgrp` 切换）

## 🔑 关键概念

- **GID**：群组标识
- **用户组**：权限管理的集合
- **effective UID/GID**：实际生效的身份
