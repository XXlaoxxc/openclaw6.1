# 第3章：主机规划与磁盘分区

## 📖 核心知识点

### 3.1 计算机硬件组成
```
CPU → 运算中心
内存 → 临时数据存储（断电丢失）
硬盘 → 永久数据存储
主板 → 连接各部件
网卡 → 网络通信
```

### 3.2 磁盘分区概念
- **MBR**：主引导记录，最多4个主分区
- **GPT**：GUID 分区表，支持更大容量和更多分区
- **分区类型**：主分区、扩展分区、逻辑分区

### 3.3 Linux 分区方案
| 挂载点 | 建议大小 | 说明 |
|--------|----------|------|
| /boot | 500MB | 启动分区 |
| / | 20GB+ | 根分区 |
| /home | 剩余空间 | 用户数据 |
| swap | 内存1-2倍 | 交换分区 |

### 3.4 FHS 标准（文件系统层次结构）
```
/           根目录
├── /bin    基本命令
├── /boot   启动文件
├── /dev    设备文件
├── /etc    配置文件
├── /home   用户家目录
├── /lib    库文件
├── /opt    第三方软件
├── /root   root家目录
├── /tmp    临时文件
├── /usr    用户程序
└── /var    可变数据
```

## 🔑 关键概念

- **挂载（mount）**：将分区关联到目录树
- **文件系统**：ext4、xfs、btrfs 等
- **inode**：文件元数据存储单元
