# 第1章：Linux 是什么

## 📖 核心知识点

### 1.1 Linux 的历史
- 1991年，Linus Torvalds 创建 Linux 内核
- 基于 Minix 系统启发
- GNU 计划 + Linux = 完整的自由操作系统

### 1.2 Linux 与 Unix 的关系
- Unix：1969年 AT&T 贝尔实验室开发
- Linux：类 Unix 系统，但非 Unix 分支
- POSIX 标准：规范 Unix-like 系统接口

### 1.3 Linux 发行版
| 发行版 | 特点 | 适用场景 |
|--------|------|----------|
| CentOS/RHEL | 稳定、企业级 | 服务器 |
| Ubuntu | 易用、社区活跃 | 桌面/服务器 |
| Debian | 稳定、自由软件 | 服务器 |
| Fedora | 新技术、激进 | 开发者 |
| Arch Linux | 滚动更新、极简 | 高级用户 |

### 1.4 Linux 的核心组成
```
Linux = 内核（Kernel）+ 系统工具 + 应用程序
         ↓
    GNU 工具链（gcc、glibc、coreutils）
```

## 🔑 关键概念

- **开源（Open Source）**：源代码公开，可自由修改分发
- **自由软件（Free Software）**：强调用户自由，非免费
- **GPL 协议**：GNU 通用公共许可证，保证软件自由

## 💡 常见误区

- ❌ Linux = Ubuntu（Ubuntu 只是发行版之一）
- ❌ Linux 是 Unix 的盗版（Linux 是独立开发的）
- ❌ Linux 只能用于服务器（桌面、嵌入式、手机都有）
