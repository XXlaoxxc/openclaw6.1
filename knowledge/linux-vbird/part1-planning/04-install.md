# 第4章：安装 CentOS 7

## 📖 核心知识点

### 4.1 安装流程
```
1. 下载 ISO → 官网/镜像站
2. 制作启动盘 → dd、Rufus、UltraISO
3. BIOS 设置 → U盘启动
4. 分区配置 → 按规划分区
5. 软件选择 → 最小安装/带GUI
6. 网络配置 → 设置IP、网关
7. 设置密码 → root 和普通用户
8. 安装完成 → 重启
```

### 4.2 安装模式
| 模式 | 说明 |
|------|------|
| 图形安装 | 默认，鼠标点击 |
| 文本安装 | 无GUI，用键盘 |
| 网络安装 | 从网络获取安装包 |
| 自动安装 | Kickstart 自动化 |

### 4.3 多重引导
- **GRUB2**：引导加载程序
- 可共存：Windows + Linux 双系统
- 注意安装顺序：先 Windows 后 Linux

## 🔑 关键概念

- **ISO 镜像**：系统安装文件
- **Live CD**：可直接运行的安装盘
- **Kickstart**：自动化安装配置文件
