# 第2章：如何学习 Linux

## 📖 核心知识点

### 2.1 学习心态
- **动手实践**：光看不练等于没学
- **不怕犯错**：虚拟机里随便折腾
- **循序渐进**：基础不牢，地动山摇

### 2.2 学习资源
| 资源类型 | 推荐 |
|----------|------|
| 官方文档 | man pages、info |
| 书籍 | 鸟哥私房菜、Linux 命令行大全 |
| 网站 | linux.vbird.org、linuxtoy.org |
| 社区 | StackOverflow、知乎 Linux 话题 |

### 2.3 实验环境搭建
- **虚拟机**：VirtualBox、VMware
- **云服务器**：阿里云、腾讯云（按量付费便宜）
- **WSL**：Windows 子系统 Linux（Win10+）
- **Docker**：轻量级容器化环境

### 2.4 学习方法
```
1. 先装系统 → 熟悉安装流程
2. 学基本命令 → cd、ls、cp、mv、rm
3. 学权限管理 → chmod、chown
4. 学 Shell → bash 基础语法
5. 学脚本 → shell script 自动化
6. 学服务 → 搭建 Web、DNS、FTP 等
```

## 🔑 关键概念

- **man 命令**：在线手册，最重要的求助工具
- **Tab 补全**：命令和文件名补全，提高效率
- **--help**：命令自带帮助信息
