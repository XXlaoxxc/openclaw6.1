# 第11章：认识与学习 Bash

## 📖 核心知识点

### 11.1 Shell 是什么
- Shell 是用户与内核之间的"翻译官"
- Bash = Bourne Again Shell，Linux 默认 Shell
- 作用：接收用户命令 → 调用程序 → 返回结果

### 11.2 Bash 基本特性
| 特性 | 说明 |
|------|------|
| Tab 补全 | 命令、文件名、路径自动补全 |
| 命令历史 | `history` 查看，`!n` 执行第 n 条 |
| 别名 | `alias ll='ls -al'` |
| 通配符 | `*` 任意字符，`?` 单个字符 |
| 管道 | `|` 前一个输出作为后一个输入 |
| 重定向 | `>` 覆盖，`>>` 追加 |

### 11.3 通配符
| 通配符 | 含义 | 示例 |
|--------|------|------|
| `*` | 任意长度字符 | `*.txt` 所有 txt 文件 |
| `?` | 单个字符 | `file?.txt` |
| `[abc]` | 字符集合 | `file[12].txt` |
| `[a-z]` | 字符范围 | `file[a-z].txt` |

### 11.4 数据流重定向
```bash
# 标准输出重定向
ls > file.txt       # 覆盖写入
ls >> file.txt      # 追加写入

# 标准错误重定向
ls 2> error.txt     # 错误输出到文件
ls 2>&1 > all.txt   # 正确和错误合并

# 标准输入重定向
sort < file.txt     # 从文件读取输入
```

### 11.5 环境变量
```bash
echo $PATH          # 命令搜索路径
echo $HOME          # 家目录
echo $USER          # 当前用户
export VAR=value    # 设置环境变量
```

## 🔑 关键概念

- **Shell 提示符**：`$` 普通用户，`#` root
- **命令执行顺序**：alias → 内置命令 → PATH 搜索
