# 《鸟哥的Linux私房菜》基础学习篇（第四版）

> 作者：鸟哥（蔡德明）| 出版社：人民邮电出版社 | 2018年

## 📖 简介

Linux 入门经典之作，被誉为"Linux入门圣经"。全书由浅入深，从基础概念到系统管理，适合初学者系统学习 Linux。

## 🗺️ 学习路线

```
第一部分 → 第二部分 → 第三部分 → 第四部分 → 第五部分
 规划安装    文件目录     Shell      用户管理    系统管理
```
